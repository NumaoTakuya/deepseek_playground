# Security Policy

## Supported Versions

This project currently supports the latest version available on the `main` branch.

Older versions or forks may not receive security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead, report it privately using GitHub's **Private Vulnerability Reporting** feature.

If that is not available, you may also contact:

numaothe@gmail.com

Please include:
- A clear description of the vulnerability
- Steps to reproduce the issue
- Potential impact

I will review the report and respond as soon as possible.
If confirmed, a fix will be prepared before public disclosure.
